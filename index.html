    About

    This plugin/channel creates a quick way to access the stream at cnreboot.com natively in Plex.

    Usage

    Tested on Plex Media Server 0.9.9.14 on Windows 7. Requires Transcoding to be ON.
    Clients Tested included Plex Media Center, RasPlex, Roku, and iOS

    Install

    To install, rename the encasing top level folder "cnreboot.bundle", place it it one of the following folders depending on your OS, then restart Plex Media Server.

    • Windows Vista/7/8/8.1 or later: Copy cnreboot.bundle to C:\Users*your username*\AppData\Local\Plex Media Server\Plug-ins

    • Windows XP: Copy cnreboot.bundle to C:\Documents and Settings*your username*\Local Settings\Application Data\Plex Media Server\Plug-ins

    • Mac/Linux: Copy cnreboot.bundle to ~/Library/Application Support/Plex Media Server/Plug-ins
